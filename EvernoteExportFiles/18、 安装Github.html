Github                        
                        

    gitHub是一个面向开源及私有软件项目的托管平台，因为只支持git 作为唯一的版本库格式进行托管，故名gitHub。
    gitHub于2008年4月10日正式上线，除了git代码仓库托管及基本的 Web管理界面以外，还提供了订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。
    目前，其注册用户已经超过350万，托管版本数量也是非常之多，其中不乏知名开源项目 Ruby on Rails、jQuery、python 等。



Linux安装Github:


1、注册账号：
   地址： github.com




2、创建仓库
        点击头像 选择  Your profile



        选择Repositories  再点击new 
选择参数 创建

停留在这别动





3、生成ssh key

        ssh-keygen -t rsa -C "github邮箱"  (不加引号)

 一直回车

cd ~/.ssh/

cat id_rsa.pub

成功生成





4.把内容粘贴到github网站



标题自拟



添加成功



测试：
ssh -T git@github.com

5.安装git
sudo apt-get install git




6、git init 初始化仓库
     首先初始化一个本地仓库 
    git init         >_第一次创建本地仓库的时候，使用一次，再后面就不用了
    

    然后
git remote add origin git@github.com:skyscience/Python_test.git   >-这里填刚才创建仓库的






7.上传至Github
首先输入你的邮箱和你的github昵称
git config --global user.email "you@example.com”  
git config --global user.name "Your Name"


上传项目里的所有文件  （也可以使用git add +项目里的文件名）
git add .

执行git commit命令执行本次提交的变更
git commit -m "name"    >-name上传名称自定


此时工作区已经干净，没有要提交的文件了，
最后使用git push来发布本地的提交操作，
输入github帐号和密码。这样提交就完成了。
git push origin master
